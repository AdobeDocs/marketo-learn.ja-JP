# 対話型ランディングページ

数回クリックするだけで、Dynamic Chat対話型フローを任意のランディングページ（Marketoランディングページまたはカスタム）にインストールし、リンクやボタンを変換して、対話型エンゲージメントをトリガーできます。

ロックを解除できるユースケース -

- 訪問者がミーティングを予約できるようにします
- 選定に関する質問
- 訪問者の選択や興味にもとづいておすすめのコンテンツ
- ドキュメントの共有やエンゲージメントの追跡
